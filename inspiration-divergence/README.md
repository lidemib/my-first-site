# 💡 灵感发散器 | Inspiration Divergence

> 给一个关键词，还你一个灵感宇宙 ✨

一个纯前端的创意思维发散工具，通过6种专业发散方法帮你打破思维定式，从不同角度点燃灵感。

## 🎯 功能特色

### 6种发散模式

| 模式 | 说明 |
|------|------|
| 🔄 逆向思维 | 反过来想，拥抱反面，找到被忽视的机会 |
| 🔀 随机联想 | 随机词语碰撞，产生意外连接 |
| 🧩 SCAMPER | 7种创新操作法，系统化拆解重组 |
| 🌊 跨界碰撞 | 跨学科视角注入，催生跨界创新 |
| 💥 极端假设 | 推到极限，限制逼出最优解 |
| 🎭 角色换位 | 换一双眼睛看问题，世界完全不同 |

### 核心功能

- **一键发散**：输入关键词，瞬间生成多角度灵感
- **灵感炸弹**：随机主题+全模式发散，适合不知道写什么的时候
- **灵感碰撞**：选择两个灵感进行交叉融合，产生1+1>2的效果
- **收藏系统**：本地存储收藏的灵感，随时回顾
- **导出功能**：一键导出为 Markdown 文件

## 🚀 使用方式

### 在线访问

直接打开 `index.html` 即可使用，无需任何安装和配置。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/lidemib/inspiration-divergence.git

# 打开页面
cd inspiration-divergence
open index.html  # macOS
# 或
start index.html  # Windows
```

### GitHub Pages 部署

1. Fork 或推送此仓库到你的 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选择 `/ (root)`
4. 访问 `https://你的用户名.github.io/inspiration-divergence/`

## 🛠 技术栈

- 纯 HTML / CSS / JavaScript，零依赖
- LocalStorage 持久化收藏数据
